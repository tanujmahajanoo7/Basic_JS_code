# Chapter 7: DOM Query Selectors and Properties

This chapter delves deeper into DOM manipulation using query selectors and exploring DOM properties.

## Topics Covered

- **Query Selectors**: Advanced usage of querySelector and querySelectorAll
- **DOM Traversal**: Accessing parent, child, and sibling elements
- **Element Properties**:
  - `tagName` - Element tag name
  - `children` - Child elements
  - `innerText` - Text content
  - `innerHTML` - HTML content
  - `textContent` - All text including hidden elements
- **Style Manipulation**: Changing element styles via JavaScript

## Files

- `index.html` - HTML structure with various elements
- `script.js` - DOM query selector and property examples
- `style.css` - Stylesheet for the page

## Key Concepts

- Using CSS selectors with querySelector
- Navigating the DOM tree
- Understanding the differences between innerText, innerHTML, and textContent
- Direct style property manipulation

## How to Run

1. Open `index.html` in a web browser
2. Open the browser's developer console (F12)
3. Explore DOM elements and their properties in the console
4. Observe the page structure and element relationships
