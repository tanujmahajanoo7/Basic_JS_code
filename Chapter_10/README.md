# Chapter 10: Rock Paper Scissors Game

This chapter implements an interactive Rock Paper Scissors game against the computer.

## Topics Covered

- **Random Generation**: Using Math.random() for computer choices
- **Game Logic**: Implementing Rock Paper Scissors rules
- **Score Tracking**: Maintaining user and computer scores
- **DOM Updates**: Dynamically updating game UI and messages
- **Visual Feedback**: Color-coded messages for win/loss/draw

## Files

- `index.html` - HTML structure for the game interface
- `script.js` - Game logic, computer choice generation, and score tracking
- `style.css` - Styling for the game
- `images/` - Game images (rock, paper, scissors)

## Key Concepts

- Random number generation and array selection
- Conditional logic for game rules
- Score management and display
- Event-driven game flow
- Dynamic DOM updates based on game results

## How to Run

1. Open `index.html` in a web browser
2. Click on Rock, Paper, or Scissors to make your choice
3. Computer randomly selects its choice
4. View the result and updated score
5. Continue playing to see who wins more rounds
