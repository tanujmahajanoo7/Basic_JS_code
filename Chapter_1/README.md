# Chapter 1: Data Types, Objects, and Classes

This chapter introduces fundamental JavaScript concepts including data types, objects, and basic class syntax.

## Topics Covered

- **Data Types**: Understanding different JavaScript data types (String, Number, Boolean, null, undefined)
- **Variables**: Declaration and usage of variables
- **Objects**: Creating and manipulating JavaScript objects
- **Classes**: Introduction to class syntax in JavaScript

## Files

- `index.html` - HTML structure
- `script.js` - Main JavaScript code demonstrating data types, objects, and classes
- `Inst_problrm.js` - Instance problem exercises

## Key Concepts

- Primitive data types (string, number, boolean, null, undefined)
- Object creation and property access
- Object property modification
- Basic class declaration

## How to Run

1. Open `index.html` in a web browser
2. Open the browser's developer console (F12)
3. View the console output to see the results of various operations
