# Chapter 13: Fetch API and Currency Converter

This chapter demonstrates working with external APIs using the Fetch API and builds a complete currency converter application.

## Topics Covered

- **Fetch API**: Making HTTP requests to external APIs
- **Async/Await**: Modern asynchronous JavaScript syntax
- **Promises**: Understanding promise-based API calls
- **API Integration**: Fetching data from currency exchange rate APIs
- **DOM Manipulation**: Dynamic UI updates based on API responses
- **Error Handling**: Handling API errors gracefully

## Files

- `index.html` - HTML structure for the currency converter
- `script.js` - Fetch API examples and demonstrations
- `CurrencyConerter/` - Complete currency converter project

## Key Concepts

- Using fetch() to make API requests
- Processing JSON responses
- Async/await syntax for cleaner async code
- Promise chaining with .then()
- Building a complete web application with API integration

## How to Run

1. Open `index.html` in a web browser to see Fetch API examples
2. For the currency converter, open `CurrencyConerter/index.html`
3. Open the browser's developer console (F12) to see API responses
4. Test the currency converter with different currencies
