# Chapter 5: Functions and Array Higher-Order Methods

This chapter covers function declarations, arrow functions, and powerful array methods for data manipulation.

## Topics Covered

- **Function Declarations**: Traditional function syntax
- **Arrow Functions**: Modern ES6 arrow function syntax
- **Function Parameters**: Passing arguments to functions
- **Array Higher-Order Methods**:
  - `forEach()` - Iterate over array elements
  - `map()` - Transform array elements
  - `filter()` - Filter array based on condition
  - `reduce()` - Accumulate values (sum, product, etc.)

## Files

- `index.html` - HTML structure
- `script.js` - Function examples and array method demonstrations

## Key Concepts

- Function declarations vs arrow functions
- Using forEach for side effects
- Using map to transform data
- Using filter to select elements
- Using reduce for aggregation operations
- Calculating sums and products with reduce

## How to Run

1. Open `index.html` in a web browser
2. Open the browser's developer console (F12)
3. Some examples use `prompt()` for user input
4. View function outputs and array method results in the console
