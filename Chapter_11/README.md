# Chapter 11: Classes, Inheritance, and Error Handling

This chapter covers object-oriented programming concepts in JavaScript including classes, inheritance, and error handling.

## Topics Covered

- **Objects**: Creating objects with methods
- **Prototypes**: Understanding prototype-based inheritance
- **Classes**: ES6 class syntax
- **Constructors**: Initializing class instances
- **Inheritance**: Extending classes with inheritance
- **Super Keyword**: Calling parent class constructors
- **Error Handling**: Try-catch blocks for error management

## Files

- `index.html` - HTML structure
- `script.js` - Object-oriented programming examples
- `problem.js` - Additional problem exercises

## Key Concepts

- Prototype-based inheritance using `__proto__`
- Class declarations and instance creation
- Constructor functions in classes
- Extending classes and inheritance chains
- Using super() to call parent constructors
- Error handling with try-catch blocks

## How to Run

1. Open `index.html` in a web browser
2. Open the browser's developer console (F12)
3. View class instantiations, inheritance examples, and error handling demonstrations
