# Chapter 3: Loops and String Methods

This chapter introduces different types of loops in JavaScript and essential string manipulation methods.

## Topics Covered

- **For Loop**: Traditional for loop syntax
- **While Loop**: While loop iteration
- **Do-While Loop**: Do-while loop with post-condition check
- **For-Of Loop**: Iterating over strings and arrays
- **For-In Loop**: Iterating over object properties
- **String Methods**: Template literals, escape characters, and string manipulation

## Files

- `index.html` - HTML structure
- `script.js` - Main loop examples and demonstrations
- `string.js` - String methods and operations
- `problem.js` - Problem exercises

## Key Concepts

- Different loop types and when to use each
- Iterating over strings character by character
- Accessing object properties using for-in loop
- Template literals for string interpolation
- String methods: toUpperCase(), toLowerCase(), trim(), slice(), replace(), replaceAll(), charAt()

## How to Run

1. Open `index.html` in a web browser
2. Open the browser's developer console (F12)
3. View loop iterations and string operations in the console
