# Chapter 4: Arrays and Array Methods

This chapter focuses on arrays in JavaScript and the various methods available for array manipulation.

## Topics Covered

- **Array Basics**: Creating arrays, accessing elements, modifying arrays
- **Array Iteration**: Using for-of loop with arrays
- **Array Methods**:
  - `push()` - Add element at end
  - `pop()` - Remove element from end
  - `unshift()` - Add element at beginning
  - `shift()` - Remove element from beginning
  - `toString()` - Convert array to string
  - `concat()` - Merge arrays
  - `slice()` - Extract portion of array
  - `splice()` - Add/remove elements at specific index
