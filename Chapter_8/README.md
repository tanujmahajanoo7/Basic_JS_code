# Chapter 8: Event Handling and Event Listeners

This chapter covers handling user interactions through events and event listeners in JavaScript.

## Topics Covered

- **Event Handlers**: Using onclick property for event handling
- **Event Listeners**: addEventListener() method for multiple handlers
- **Event Object**: Understanding event properties (type, target, clientX, clientY)
- **Event Removal**: removeEventListener() for cleaning up handlers
- **Practical Examples**: 
  - Toggle button for dark/light mode
  - Using classList for styling

## Files

- `index.html` - HTML structure with buttons and interactive elements
- `script.js` - Event handling examples and implementations
- `style.css` - Stylesheet with dark/light mode styles

## Key Concepts

- Difference between inline event handlers and event listeners
- Adding multiple event listeners to the same element
- Event object and its properties
- Practical toggle functionality implementation
- Using classList API for DOM manipulation

## How to Run

1. Open `index.html` in a web browser
2. Click the buttons to see event handlers in action
3. Try the dark/light mode toggle buttons
4. Open the browser's developer console (F12) to see event logs
