# Basic JavaScript Code

This repository contains a comprehensive collection of JavaScript learning materials organized by chapters. Each chapter covers different fundamental concepts and practical implementations of JavaScript.

## Overview

This project is structured as a progressive learning path through JavaScript fundamentals, from basic data types to advanced concepts like async programming and API integration.

## Chapters

- **Chapter 1**: Data Types, Objects, and Classes
- **Chapter 2**: Operators and Conditional Statements
- **Chapter 3**: Loops and String Methods
- **Chapter 4**: Arrays and Array Methods
- **Chapter 5**: Functions and Array Higher-Order Methods
- **Chapter 6**: DOM Basics and Manipulation
- **Chapter 7**: DOM Query Selectors and Properties
- **Chapter 8**: Event Handling and Event Listeners
- **Chapter 9**: Tic Tac Toe Game Project
- **Chapter 10**: Rock Paper Scissors Game
- **Chapter 11**: Classes, Inheritance, and Error Handling
- **Chapter 12**: Asynchronous JavaScript and Callbacks
- **Chapter 13**: Fetch API and Currency Converter Project

## Getting Started

1. Navigate to any chapter folder
2. Open the `index.html` file in a web browser
3. Check the browser console for output
4. Review the `script.js` file for implementation details

## Structure

Each chapter typically contains:
- `index.html` - HTML structure
- `script.js` - JavaScript implementation
- `style.css` - Stylesheet (if applicable)
- Additional files for problems and exercises

## Prerequisites

- Basic understanding of HTML and CSS
- A modern web browser with developer console enabled
- A code editor (recommended)
