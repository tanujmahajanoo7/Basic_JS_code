# Chapter 9: Tic Tac Toe Game

This chapter demonstrates building an interactive Tic Tac Toe game using DOM manipulation and event handling.

## Topics Covered

- **Game Logic**: Implementing Tic Tac Toe win conditions
- **DOM Manipulation**: Creating interactive game board
- **Event Handling**: Handling player clicks on game boxes
- **State Management**: Tracking game state and player turns
- **Game Features**:
  - Win detection
  - Draw detection
  - Reset functionality
  - New game functionality

## Files

- `index.html` - HTML structure for the game board
- `script.js` - Complete game logic and event handling
- `style.css` - Styling for the game interface

## Key Concepts

- Managing game state (turn tracking, win detection)
- Implementing win patterns (rows, columns, diagonals)
- Disabling elements after game end
- Creating a complete interactive game experience

## How to Run

1. Open `index.html` in a web browser
2. Click on the boxes to make your move
3. Players alternate between X and O
4. Game announces winner or draw
5. Use "New Game" or "Reset" buttons to start over
